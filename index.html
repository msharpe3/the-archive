Movie Lookup

Barcode and title lookups use a free movie database as a backup source. Get a free key (instant, no card needed) at omdbapi.com/apikey.aspx, then paste it below. One-time setup per device.

trailers & better posters (recommended)

Adding a free key from themoviedb.org/settings/api gets you higher-res posters, better descriptions, and trailer links, all in one lookup. If it doesn't find a match, the key above is used as a backup.

backup barcode source (optional)

Barcode scans try one free product database first. Adding a free key from eandata.com gives it a second source to try if the first one misses — worth adding if scans are coming up empty often.
